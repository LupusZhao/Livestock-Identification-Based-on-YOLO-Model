# YOLO训练目标检测模型

python版本：3.9
建议使用anaconda来建一个虚拟环境，创建虚拟环境的同时在里面安装指定的python版本，
然后在虚拟环境中安装下面的依赖库【注意：要先激活虚拟环境，再安装依赖库】
建议用pycharm来打开项目，并在pycharm中使用建好的虚拟环境运行项目文件
关于虚拟环境的创建 和 如何在pycharm中使用建好的虚拟环境，请自行参考网上教程

（一）安装torch, 根据自己的系统选择安装CPU版本或GPU版本【有独立显卡则安装GPU版本，没有则安装CPU版本】
    安装CPU版本torch
    ```
    pip install torch==2.0.1+cpu torchvision==0.15.2+cpu -f https://download.pytorch.org/whl/torch_stable.html
    ```
    安装GPU版本torch（如果GPU版无法安装成功，可直接安装CPU版的）
    ```
    pip install torch==2.0.1+cu118 torchvision==0.15.2+cu118 -f https://download.pytorch.org/whl/torch_stable.html
    ```

（二） 安装其他依赖库，(需要具体指定requirements.txt的位置【此文件就在项目中】，
    例如，requirements.txt文件在C:\Users\2023\Desktop\requirements.txt，
    则运行以下命令：pip install -r "D:\浏览器下载\livestock\livestock\livestock_detection\requirements.txt"
    ```
    pip install -r requirements.txt
    ```
（三）模型训练
    1. 修改数据集路径：
    打开data/data.yaml, 修改train和val的路径为自己数据集的路径【原项目数据集在datasets结尾的压缩包中，可自行解压】

    2. 模型训练命令【原项目模型已经训练好了，可跳过此步骤】：
    在pycharm或者vscode中运行train.py文件

    3. 模型验证命令【原项目模型已经训练好了，可跳过此步骤】：
    1）修改val.py中模型路径为自己训练好的模型路径，原项目模型路径在runs/detect/train/weights/best.pt
    2）运行val.py文件

（四）模型应用【原项目模型已经训练好了，直接运行gui.py文件即可弹出界面】：
    1. 修改gui.py中最下面的模型路径为自己训练好的模型路径，原项目模型路径在runs/detect/train/weights/best.pt
    2. 打开gui.py文件并运行此文件即可使用训练好的模型

（五）注意事项：
    1. 如果界面不居中显示，可根据ui.py的MainWindow.resize(692, 732)和self.frame.setGeometry(QRect(0, 0, 691, 731))配合调整
    2. 可修改界面的字体大小，在ui.py的最下面修改
    3. 项目运行有问题请及时联系
